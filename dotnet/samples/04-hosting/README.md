# Hosting & Protocol Samples (.NET)

Deploy your agents using different hosting patterns and protocols.
These are multi-project solutions with their own `.csproj` files.

| Pattern | Sample | Description |
|---------|--------|-------------|
| A2A Protocol | [a2a/](./a2a/) | Agent-to-Agent protocol (client + server) |
| AG-UI Protocol | [ag-ui/](./ag-ui/) | Agentic UI protocol (client + server) |
| Azure Functions | [azure-functions/](./azure-functions/) | Serverless hosting with Durable Functions |
| OpenAI Endpoints | [openai-endpoints/](./openai-endpoints/) | OpenAI-compatible hosted endpoints |

For docs: https://learn.microsoft.com/agent-framework/integrations/overview
